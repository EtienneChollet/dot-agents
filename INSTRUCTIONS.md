# User Instructions

These are your global instructions, deployed to all harnesses as `CLAUDE.md`
(Claude Code) and `AGENTS.md` (Pi). Edit this file to customize agent behavior
across all projects.

## Getting Started

Replace this content with your own preferences and conventions. Common things
to include:

- **Behavioral rules**: How you want the agent to approach tasks (e.g., always
  investigate before implementing, always ask before committing)
- **Tool preferences**: Package managers, test runners, linters you use
- **Code style**: Naming conventions, import style, type hint preferences
- **Git conventions**: Branch naming, commit message format, PR workflow
- **Project-specific context**: Machine names, cluster details, team conventions
